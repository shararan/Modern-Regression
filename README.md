# Modern Regression — Independent Study

Python implementation of programming assignments from CMU's Modern Regression course (2017), completed independently as self-study.

## Contents

### Homework Assignments (hw3–hw9)
Solved programming exercises 3–9 of the course, covering:
- Exploratory Data Analysis (EDA)
- Ordinary Least Squares (OLS), Weighted Least Squares (WLS) regression
- Model diagnostics and residual analysis
- Variable selection and model comparison

### Independent Modelling Project (`EDA.ipynb`)
Applied regression and EDA techniques to the Nanook Quantitative Modelling challenge as an additional self-directed modelling exercise.

## Tools & Libraries
- Python, Jupyter Notebook
- pandas, numpy, matplotlib, statsmodels

## Note
This work was completed independently as self-study — I am not and was not enrolled at CMU.
